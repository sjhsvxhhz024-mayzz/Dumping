# global-metadata.dat — раскладка заголовка (Unity il2cpp v39, Oxide-сборка)

Извлечено детерминированно из `libil2cpp.so`, функция-загрузчик `sub_4D765C8`
(xref на строку `global-metadata.dat` @ vaddr `0x1DFF376`, ADRL @ `0x4D765E0`).
Дизассемблер: capstone ARM64, окно `0x4D765C8..0x4D87600`.

## Шифрование
- Ключ XOR: `0xA5C3F19D`.
- Зашифрован весь заголовок в диапазоне `[0x08, 0x17C)` (uint32-поля).
- `magic` (`0x00` = `0xFAB11BAF`) и `version` (`0x04` = `39`) — плейнтекст.
- Данные секций (после `0x17C`) — плейнтекст.
- Декриптор: `tools/decrypt_metadata.py` (проверено: 0 оффсетов за EOF, 0 полей ≥ 2^31).
